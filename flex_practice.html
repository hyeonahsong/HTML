1. container 속성
1). 중심축을 기준으로 한 속성
/* 가로축을 기준으로 줄 세우는 속성 */
display: flex;
/* 중심축을 기준으로 요소들을 중간으로 배치*/
justify-content: center;
/* 중심축을 기준으로 요소들을 오른쪽으로 배치 */
justify-content: right;
/* 일정한 간격으로 요소들을 배치 */
justify-content: space-around;
/* 요소를 반대로 정렬 */
flex-direction: row-reverse;


2. 반대축을 기준으로 한 속성
/* 중심축을 반대축인 세로로 바꾸는 속성 */
flex-direction: column;
/* 반대축을 기준으로 하여 가운데 정렬하는 속성 */
align-items: center;
